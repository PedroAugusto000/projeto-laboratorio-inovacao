ARRUMANDO CÓDIGO:
Colaborador na view:
1. buscar_colaboradores.php -> Busca funcionários na hora de inserir o Cozinheiro e o Degustador nas receitas
2. colaborador_funcionarios.php -> Aba do CRUD de funcionarios
3. colaborador_inicial.php -> Aba colaborador inicial logo após clicar em "Aba do colaborador" no index.php
4. editar_colaborador.php -> Aba de edição de funcionários logo após o registro
5. registro_colaborador.php -> Aba de registro de funcionário

Livro na view:
1. detalhe_livro.php -> Aba que aparece logo após clicar em um livro específico
2. editar_livro.php -> Aba de edição de livro
3. gerir_livros.php -> Aba do CRUD de livros
4. registro_livro.php -> Aba de registrar os livros
5. todos_livros.php -> Tela de todos os livros registrados do projeto

Receitas na view:
1. detalhe_receita.php -> Aba que aparece logo após clicar em uma receita específica
2. editar_receitas.php -> Tela de editar as receitas
3. gerenciar_receitas.php -> Aba do CRUD de receitas
4. registro_receita.php -> Aba de registrar as receitas
5. todas_receitas.php -> Tela de todas as receitas registrados do projeto

Outros:
navbar.php -> Barra de navegação padrão 
login.php -> Tela de login
